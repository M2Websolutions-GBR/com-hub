# Frontend Project

This is the frontend project for Com-Hub. To get started, follow these steps:

## Installation

First, navigate to the `frontend` directory:

```bash
cd frontend

npm install
```

## Start Webserver

```bash
npm run dev
```

## Additional Commands

Build:

```bash
npm run build
```
